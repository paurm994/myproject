HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Utiliza etiquetas para definir la estructura y el contenido de una página web, incluidos elementos como texto, imágenes, enlaces y otros elementos multimedia.
